### 📝 Опис проєкту
Інформаційна система **«Довідник астронома»** призначена для систематизації даних про небесні тіла (зірки, планети, туманності) та їх приналежність до сузір'їв. Програма дозволяє користувачеві керувати базою астрономічних об'єктів, здійснювати пошук та зберігати результати.

### 🚀 Основний функціонал
- [x] **Керування даними:** Додавання, редагування та видалення записів про об'єкти.
- [x] **Класифікація:** Зв'язок між небесними тілами та сузір'ями (ООП зв'язки).
- [x] **Пошук та фільтрація:** Пошук об'єктів за назвою, яскравістю або типом.
- [x] **Робота з файлами:** Збереження та завантаження бази даних (JSON/XML).
